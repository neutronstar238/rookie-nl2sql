## Few-Shot 示例

### 示例 1: 简单查询
问题: 查询所有客户
SQL:
```sql
SELECT * FROM Customer;
```

### 示例 2: 条件查询  
问题: 查询来自巴西的客户
SQL:
```sql
SELECT * FROM Customer WHERE Country = 'Brazil';
```

### 示例 3: 统计查询
问题: 统计每个国家的客户数量
SQL:
```sql
SELECT Country, COUNT(*) as customer_count
FROM Customer
GROUP BY Country;
```

### 示例 4: 排序查询
问题: 查询销售额最高的前10个客户
SQL:
```sql
SELECT c.FirstName, c.LastName, SUM(i.Total) as total_amount
FROM Customer c
JOIN Invoice i ON c.CustomerId = i.CustomerId
GROUP BY c.CustomerId, c.FirstName, c.LastName
ORDER BY total_amount DESC
LIMIT 10;
```

### 示例 5: 范围查询
问题: 查询价格在0.99到1.99之间的歌曲
SQL:
```sql
SELECT Name, UnitPrice
FROM Track
WHERE UnitPrice BETWEEN 0.99 AND 1.99;
```

### 示例 6: 聚合统计
问题: 统计总订单数
SQL:
```sql
SELECT COUNT(*) as total_orders FROM Invoice;
```

### 示例 7: 条件过滤
问题: 查询金额大于10的发票
SQL:
```sql
SELECT * FROM Invoice WHERE Total > 10;
```

### 示例 8: 产品排序
问题: 按价格降序排列歌曲
SQL:
```sql
SELECT Name, UnitPrice
FROM Track
ORDER BY UnitPrice DESC;
```
